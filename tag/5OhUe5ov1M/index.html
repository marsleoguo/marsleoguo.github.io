2017-07-19

dpkg 是Debian package的简写，为”Debian“ 操作系统 专门开发的套件管理系统，用于软件的安装，更新和移除。

所有源自"Debian"的Linux的发行版都使用 dpkg, 例如"Ubuntu"

安装软件

命令:dpkg -i <.deb file name>

实例: dpkg -i ~/Download/mozybackup_i386.deb

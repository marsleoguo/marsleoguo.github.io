tomcat添加到服务

2017-08-05

1.将zip版的tomcat解压:我自己的是解压到E:\tomcat-1(这个根据自己的情况而定)

2.修改bin文件夹下的startup.bat,在第一行加入以下两句话:

set JAVA_HOME=F:\Program Files\Java\jdk1.6.0_04(自己机器上的jdk安装路径)
set CATALINA_HOME=E:\tomcat-1(自己解压的tomcat的位置)

同样修改bin文件夹下的shutdown.bat.也是加入上面的那2句话。

3.然后修改bin文件夹下的service.bat，修改的地方如下:

set SERVICE_NAME=Tomcat1(这个根据自己的情况起名字,这个名字可用中net start/stop 来启动服务)
set PR_DISPLAYNAME=Apache Tomcat1(service.msc中显示的名字)

4.最后是在cmd命令窗口中:进入我们刚解压的tomcat-1\bin目录,然后执行

service.bat install (看到命令窗口中The service ‘Tomcat1’ has been installed ,就说明你安装服务成功)

如果自己想将这个服务卸掉,可以用service.bat remove 命令将其卸载。

5.到此，你就可以用net start/stop tomcat1 ,直接双击startup.bat ，shutdown.bat来直接启动或关闭服务,或者直接进去service.msc中来启动和停止服务了。

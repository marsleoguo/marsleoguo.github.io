atom 手动安装插件包

  1. 在这里搜索需要的 Package

  2. 点击report获取git链接地址，如下图：

  3. 进入home文件夹下atom的packages文件夹

cd ~/.atom/packages
  1. 使用git下载插件：git clone <你想安装的 Package 的仓库链接>
git clone https://atom.io/packages/emmet
  1. 进入插件的文件夹,如下：
cd <Package 路径> # cd emmet-atom
  1. 安装插件：
npm install
